# Basic README Generator
## **Description**

Generate a README file from command-line input.
## **Install**

```shell
npm install basic-readme-generator
```
## **Usage**

```shell
npm explore basic-readme-generator -- npm run generateREADME
```

## **Uninstall**

```shell
npm uninstall basic-readme-generator
```
